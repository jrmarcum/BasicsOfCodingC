C has no built-in map type. This lesson implements a simple string-to-int map using parallel arrays. Output is printed in insertion or sorted order to match Go's formatted output.

___
##### Run Command:

`$ zig run -lc maps.c`

##### Results:

`map: map[k1:7 k2:13]`
`v1:  7`
`len: 2`
`map: map[k1:7]`
`prs: false`
`map: map[bar:2 foo:1]`
